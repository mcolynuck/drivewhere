From the root, execute "foreman start"
Start the app using "localhost:5000"

GIT commands:
-To log into heroku:
	heroku login
-Add files to deployment:
	git add .
-Commit changes with comment:
	git commit -m "Comment here"
-Push files to Heroku platform:
	git push heroku master
-Start to make sure it deployed correctly:
	heroku open
